# 复盘性能检查清单

## 基准环境

- 设备：iPad Air M1
- 构建：Release
- 棋谱：同一份 200 手左右棋谱，优先包含提子、连续 Pass 和中后盘转换
- 状态：AI 批量分析完成后再开始拖动测试

## 拖动测试

- 地盘归属关闭，连续拖动进度条 5 秒，观察棋盘是否跟手、是否明显掉帧。
- 地盘归属开启，连续拖动进度条 5 秒，观察棋盘和地盘 overlay 是否卡顿。
- 快速从开局拖到终局，再拖回中盘，确认棋子、提子数、当前手和最后一手标记一致。
- 反复拖过有提子的局部，确认被提棋子能正确消失和恢复。
- 拖到连续 Pass 的最后两手，确认终局状态和当前手显示正确。

## 面板测试

- AI 分析完成后，拖动 5 秒，确认右侧复盘面板滚动不被明显阻塞。
- 连续点击“本局关键失误”列表 10 次，确认跳转准确且棋盘无明显延迟。
- 开关“显示地盘归属”各拖动 5 秒，分别记录手感差异。

## 记录项

- App 版本和 Git 提交号
- 棋谱文件名、总手数、是否包含提子和连续 Pass
- Xcode Debug Navigator 中的内存峰值
- 是否出现卡住、明显掉帧或滑块松手后棋盘才追上
- 若出现问题，记录大致手数范围和地盘归属开关状态
